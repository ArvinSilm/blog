Windows本地运行MapReduce

Windows本地运行MapReduce

Windows本地无HDFS调试MapReduce程序。

Maven

依赖

<dependency>
    <groupId>org.apache.hadoop</groupId>
    <artifactId>hadoop-client</artifactId>
    <version>2.5.1</version>
</dependency>

代码

需要修改org.apache.hadoop.io.nativeio.NativeIO和org.apache.hadoop.mapred.YARNRunner的代码。 新建org.apache.hadoop.io.nativeio和org.apache.hadoop.mapred包，将代码放入其中。

环境

本地解压一份所需版本的Hadoop源码，在系统环境变量中添加HADOOP_HOME，指向源码路径。 将winutils.exe拷贝到HADOOP_HOME/bin目录下。
